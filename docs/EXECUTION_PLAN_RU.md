## Зафиксированные решения

- **РЕШЕНИЕ-1 (печать не-ЧЗ):** конструктор не нужен (этикетки одинаковые) → у не-ЧЗ товара только поле количества ШК ВБ, без конструктора.
- **РЕШЕНИЕ-2 (множитель):** «× количество товаров в упаковке» применяется к ШК ВБ **и в упаковке, и в каталоге**.

## Правила прохождения

1. **Строго по порядку, одна задача за раз.** Между задачами — коммит.
2. Зелёный тест/сборка до перехода к следующей: backend `ruff && mypy && pytest <scope>`; frontend `npm run build && npm run test:e2e <spec>`.
3. Файл правит только текущая задача — параллельной работы по соседним ID нет (последовательность снимает коллизии).
4. Строка в `TASKLOG.md` на каждую закрытую задачу.
